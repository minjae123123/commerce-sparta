JAVA 커머스 콘솔 앱


프로젝트 소개
- JAVA 커머스 애플리케이션 (도전 과제 step2  까지 진행, step3 x)

진행 목표
- 카테고리와 상품을 반복문으로 조회
- 장바구니 담기, 주문, 주문 취소 흐름을 구현
- 관리자 인증 후 상품 추가, 수정, 삭제 기능을 구현

파일 구조
- Admin.java : 관리자 인증, 관리자 메뉴 흐름 담당
- Cart.java : 장바구니 항목 관리
- Category.java : 카테고리/상품 조회
- CommerceSystem.java : 주문 처리, 메인 메뉴 출력, 전체 흐름 제어
- Main : 프로그램 시작
- Product : 상품의 상태 정보 관리
  

구현 내용
- 기본 메인 메뉴에서 카테고리 조회 및 장바구니 추가, 장바구니 확인, 주문 취소, 관리자 모드 진입 기능 제공
- 주문 완료 시 상품 재고 차감 및 장바구니 비우기 처리
- 관리자 비밀번호 인증 후 상품 추가, 가격 수정, 설명 수정, 재고 수정, 삭제 기능 제공

실행 방법
- 콘솔 메뉴에서 카테고리를 선택해 상품을 조회
- 상품을 장바구니에 담고 주문 메뉴에서 결제를 진행
- 상품을 카테고리에 추가하거나 수정 및 삭제를 하고싶다면 관리자 모드로 진입
  
실행 환경
- Java 21
- IntelliJ IDEA

**필수 과제까지는 Category.java에서 카테고리 별로 리스트를 만들어 구현하였는데 도전과제에서의 수량 업데이트 문제를 해결할 수 없어
, 직접 Product.java 클래스에 필드의 값을 조회하고 업데이트 하기 위해
도전 과제부터는 CommerceSystem.java에 상품 리스트를 만들고 카테고리를 나누어 리스트에 넣고 구현을 하였습니다.
